Un altfel de caine utilitar

Cainele utilitar este cainele dresat pentru a indeplini o anumita sarcina, in acest fel venind in sprijinul oamenilor,acolo si atunci cand este necesar.

Exemple de caini utilitari:

  • Cainii de urma.
  • Cainii pentru detectare explozivi.
  • Cainii pentru detectare tutun.
  • Cainii de cautare-salvare.
  • Cainii pentru terapie.
  • Cainii pentru detectare droguri.
  • Cainii pentru descoperire cadavre.

Cainele pentru descoperirea cadavrelor este un caine utilitar care se afla de obicei in dotarea structurilor de politie. Acesta este folosit indeosebi pentru gasirea persoanelor disparute, in situatia unor dezastre, si pentru gasirea ramasitelor umane in cazul unor infractiuni.

Activitatea de cautare in care este folosit un caine pentru descoperire de cadavre, este influentata de stadiul de descompunere al cadavrului (mirosul emanat), dar si de conditii meteo la momentul cautarii (temperatura, vant, precipitatii).De asemenea descompunea unui cadavru este influentata de mai multi factori,respect, timp, presiune, umezeala si conditii meteo.

Etapele descompunerii unui cadavru
Descompunerea unui cadavru: Descriere:
Autoliza Celulele organismului incep sa se descompuna.
Descompunerea initiala Pielea isi schimba culoarea, sunt emanate gaze de putrefactie, apare mirosul si cadavrul se umfla
Descompunerea avansata Viermii se inmultesc,tesutul incepe sa dispara si se fac vizibile oasele.
Scheletizare Tesuturile ramase se usuca si se desprind,scheletul ramanand expus.
